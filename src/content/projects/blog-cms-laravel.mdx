# Blog CMS Laravel â€“ Pemenang Karir

Pemenang Karir adalah platform blog & CMS modern berbasis Laravel dan React Inertia yang memungkinkan user menjadi author dan menulis artikel secara terstruktur.

---

## Fitur Utama

- Author registration & approval
- Post management (draft, pending, publish)
- Category management
- Dashboard admin & author
- Email notification
- Role-based access

---

## Teknologi

- Laravel 12
- React + Inertia.js
- MySQL
- Tailwind CSS

---

## Manfaat Sistem

- Kontrol kualitas konten
- Workflow editorial jelas
- Cocok untuk platform edukasi & personal branding

---

## Pembelajaran

- CMS architecture
- Approval workflow
- Integrasi backend & SPA frontend
