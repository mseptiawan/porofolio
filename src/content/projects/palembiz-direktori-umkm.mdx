# Palembiz – Direktori UMKM Palembang



Palembiz adalah website direktori UMKM berbasis peta interaktif yang membantu pengguna menemukan UMKM di Palembang berdasarkan lokasi, kecamatan, dan kategori usaha.

---

## Latar Belakang

UMKM lokal sering kesulitan mendapatkan exposure digital. Palembiz dibuat sebagai solusi untuk menghubungkan UMKM dengan calon pelanggan secara berbasis lokasi.

---

## Fitur Utama

- Peta interaktif per kecamatan
- Filter kategori UMKM
- Fitur “Sekitar Saya”
- Detail UMKM lengkap
- Favorit UMKM
- Pagination & Search
- Dark Mode
- AI Voice (ElevenLabs Text-to-Speech)

<div className="p-4 bg-emerald-50 border-l-4 border-emerald-500 dark:bg-slate-800 my-4">
  **Highlight:** Integrasi AI Voice untuk membacakan deskripsi UMKM.
</div>

---

## Teknologi

- React
- Vite
- JavaScript
- Tailwind CSS
- Leaflet
- ElevenLabs API
- Vercel

---

## Nilai Sistem

- Membantu UMKM mendapatkan exposure digital
- UX berbasis lokasi
- Mudah dikembangkan ke versi mobile app
