# Sistem Insentif Promotor Changhong

Sistem ini dikembangkan saat magang di **PT. Changhong Electric Indonesia** untuk menggantikan proses perhitungan insentif promotor yang sebelumnya dilakukan secara manual.

---

## Permasalahan Sistem Lama

- Perhitungan insentif rawan salah
- Proses rekap memakan waktu lama
- Sulit memantau performa promotor
- Data tersebar di banyak file

---

## Solusi Sistem

Sistem berbasis web yang mampu:

- Mengelola data promotor & produk
- Menghitung insentif otomatis
- Menampilkan ranking promotor
- Rekap insentif per periode

<div className="p-4 bg-indigo-50 border-l-4 border-indigo-500 dark:bg-slate-800 my-4">
  **Info:** Sistem mendukung upload data massal via Excel.
</div>

---

## Fitur Utama

- Upload penjualan via Excel
- Model insentif per produk
- Target promotor bulanan
- Dashboard interaktif
- Role-based access (Admin, Manager, SPG)
- Rekap & laporan insentif

---

## Teknologi

- Laravel
- Blade
- MySQL
- Tailwind CSS
- Excel Import

---

## Pembelajaran

- Analisis sistem bisnis nyata
- Otomatisasi proses manual
- Validasi data skala besar
- Sistem berbasis role & approval
