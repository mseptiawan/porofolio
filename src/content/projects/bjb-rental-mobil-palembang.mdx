# Proyek Aplikasi Rental Mobil Berbasis Web

Proyek ini merupakan salah satu pengalaman paling komprehensif saya dalam membangun aplikasi bisnis end-to-end. Aplikasi rental mobil ini dirancang untuk membantu pemilik usaha rental dalam mengelola armada, transaksi penyewaan, pelanggan, serta laporan keuangan secara terintegrasi dan real-time.

Pengembangan aplikasi dilakukan melalui beberapa fase penting yang mengacu pada praktik **System Analysis & Software Development Life Cycle (SDLC)**.

---

## Latar Belakang Proyek

Banyak usaha rental mobil skala kecil hingga menengah masih menggunakan cara manual, seperti pencatatan di buku atau spreadsheet sederhana. Hal ini menimbulkan berbagai permasalahan:

- Data kendaraan sering tidak sinkron
- Kesulitan melacak status mobil (tersedia / disewa / servis)
- Risiko kehilangan data transaksi
- Laporan keuangan tidak akurat dan memakan waktu

Aplikasi ini hadir sebagai solusi digital untuk mengatasi masalah tersebut.

## Ruang Lingkup Aplikasi

Aplikasi rental mobil ini mencakup beberapa modul utama:

1. Manajemen Data Mobil
2. Manajemen Pelanggan
3. Transaksi Penyewaan & Pengembalian
4. Manajemen Pembayaran
5. Laporan & Monitoring Bisnis

Setiap modul dirancang saling terintegrasi agar alur bisnis berjalan efisien.

---

## Tahap Analisis Sistem

Tahap analisis menjadi fondasi utama dalam proyek ini. Fokus utama adalah memahami proses bisnis rental mobil secara menyeluruh.

### 1. Identifikasi Stakeholder

- Pemilik usaha rental
- Admin operasional
- Petugas lapangan
- Pelanggan

### 2. Identifikasi Masalah Sistem Lama

- Tidak ada validasi data pelanggan
- Status mobil sering tertukar
- Perhitungan denda keterlambatan tidak konsisten
- Sulit membuat laporan bulanan

### 3. Kebutuhan Sistem

**Kebutuhan Fungsional:**

- Input dan update data mobil
- Pencatatan transaksi sewa
- Perhitungan otomatis biaya sewa dan denda
- Cetak invoice dan laporan

**Kebutuhan Non-Fungsional:**

- Sistem berbasis web
- Akses berbasis role (admin & owner)
- Keamanan data
- Mudah digunakan (user-friendly)

---

## Desain Sistem

### Use Case Utama

- Admin mengelola data mobil
- Admin memproses transaksi sewa
- Sistem menghitung biaya dan durasi sewa
- Owner melihat laporan pendapatan

### Desain Data (Entity Utama)

- Mobil
- Pelanggan
- Transaksi
- Pembayaran
- User

Relasi antar data dirancang untuk meminimalkan redundansi dan memastikan integritas data.

Proyek aplikasi rental mobil ini memberikan pemahaman mendalam bahwa keberhasilan sistem tidak hanya bergantung pada teknologi, tetapi juga pada **ketepatan analisis kebutuhan dan pemahaman proses bisnis**.
