# Finalis Juara 5 – Web Development Competition

Kegiatan ini merupakan **lomba Web Development tingkat nasional** yang diselenggarakan di  
**UIN Raden Fatah Palembang** dalam rangka acara  
**“Tech Beyond Limits: Building Our Future Together – Innovation, Competition, and Entrepreneurial Excellence.”**

Tim kami berhasil masuk **finalis dan meraih Juara 5**, bersaing dengan **18 tim dari berbagai daerah di Indonesia**.

---

## Informasi Kegiatan

- **Jenis Kegiatan:** Web Development Competition
- **Nama Acara:** Tech Beyond Limits: Building Our Future Together
- **Penyelenggara:** UIN Raden Fatah Palembang
- **Skala:** Nasional
- **Jumlah Tim:** 18 Tim
- **Pencapaian:** Finalis – Juara 5

---

## Tim & Pembagian Peran

Tim kami terdiri dari **2 orang** dengan pembagian tugas yang jelas:

- **Septiawan (Saya)**  
  → Fokus pada **pengembangan backend dan API**, termasuk:

  - Perancangan struktur backend
  - Pengembangan REST API
  - Integrasi data antara backend dan frontend
  - Validasi dan alur logika sistem

- **Adit Jansa**  
  → Fokus pada **desain dan pengembangan tampilan website**, termasuk:
  - UI layout
  - User experience
  - Konsistensi visual antarmuka

Kolaborasi ini memungkinkan pengembangan sistem berjalan seimbang antara fungsionalitas dan tampilan.

---

## Hasil & Pembelajaran

- Masuk **finalis nasional** dari 18 tim
- Mendapat pengalaman nyata membangun aplikasi web secara tim
- Memahami pentingnya:
  - Sinkronisasi backend & frontend
  - Pembagian peran dalam tim kecil
  - Kualitas arsitektur sistem dan presentasi produk

Pengalaman ini memperkuat pemahaman saya dalam pengembangan web end-to-end serta kerja tim dalam kompetisi teknologi.
